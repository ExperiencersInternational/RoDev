How to enable HTTP requests on a Roblox game (Easy)

Today in this tutorial you will be learning on how to enable HTTP requests on a Roblox game. To start, the game will need to be published. To continue/finish the tutorial please view the Google Slides presentation below.

Text Tutorial

This tutorial will show you how to enable HTTP requests in your Roblox game. The game will first need to be published before you continue. Once you've got your published game open in Studio, press the Game Settings button in the home tab. After you've done that, in the window that appears on your screen go into the Security tab. Now finally turn on allow HTTP requests and click save. Congratulations, you've now enabled HTTP requests on your Roblox game!
